Some personal projects

Reactive Dog Journal

  • February 2017
  • Live on Heroku
  • GitHub

I'm building a Reactive Dog Journal in Python and the Django framework that allows a user to input daily dog-walking logs to track their dog's reactivity and aggression training progress, storing the data in a PostgreSQL database. I also make use of the Bootstrap CSS framework and integrate Highcharts.js to generate a line graph indicating reactivity trends over time. You can find it live on Heroku.

Currently only an admin user can access and create posts, but the long-term plan is to create a user database so that folks can create an account and their own reactive dog journal.

Walky Talky

  • nwHacks
  • March 2017
  • blog
  • GitHub

We made an "Uber for walking tours" Android mobile app (well... we didn't finish all the functionality we wanted). We used a CockroachDB SQL database and Node.js Express server on the backend, and pulled from Yelp and Google Maps APIs to display tour destinations and a map route to follow.
