Reactive Dog Journal

  • February 2017
  • Live on Heroku
  • GitHub

A Reactive Dog Journal in Python and the Django framework that allows a user to input daily dog-walking logs to track their dog's reactivity training progress, storing data in a PostgreSQL database. Front-end uses Bootstrap CSS and integrates Highcharts.js to generate a graph indicating reactivity trends over time. You can find it live on Heroku.

The long-term plan is to create a user database allowing secure sign-in and individual journals.

Walky Talky

  • nwHacks
  • March 2017

We made an "Uber for walking tours" Android mobile app (well... we didn't finish all the functionality we wanted). We used a CockroachDB SQL database and Node.js Express server on the backend, and pulled from Yelp and Google Maps APIs to display tour destinations and a map route to follow.
